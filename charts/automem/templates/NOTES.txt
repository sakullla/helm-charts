1. Access AutoMem inside the cluster:

  kubectl port-forward svc/{{ include "automem.fullname" . }} 8001:{{ .Values.service.port }}

2. Verify health endpoint:

  curl http://127.0.0.1:8001/health

3. Internal backing services rendered by this chart:
{{- if .Values.falkordb.enabled }}
  - FalkorDB: {{ include "automem.falkordbHost" . }}:6379 (bitnami/redis chart with FalkorDB image)
{{- else }}
  - FalkorDB: disabled (set api.falkordbHost/api.falkordbPort for external DB)
{{- end }}
{{- if .Values.qdrant.enabled }}
  - Qdrant: {{ include "automem.qdrantHost" . }}:6333 (qdrant/qdrant chart)
{{- else }}
  - Qdrant: disabled (set api.qdrantUrl for external DB)
{{- end }}

4. Ensure you set your AutoMem app image:

  --set image.repository=<your-registry>/automem --set image.tag=<your-tag>
